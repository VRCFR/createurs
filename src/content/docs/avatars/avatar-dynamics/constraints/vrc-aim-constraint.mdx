Le composant `VRCAimConstraint` fait pivoter le transform cible pour qu'il vise les sources. Vous pouvez personnaliser la direction considérée comme avant.

![](/img/avatars/constraints/aim.png)

<ConstraintMainSettings />
- **Aim Vector** - Contrôle les axes qui doivent être orientés vers les sources.
- **Up Vector** - Contrôle l'axe vers le haut pour cette contrainte. C'est l'axe du transform affecté que la contrainte essaiera d'aligner avec l'axe vers le haut du monde défini ci-dessous.
- **World Up Type** - Contrôle la direction vers le haut du monde utilisée par cette contrainte de visée.
  - **Scene Up** - Considère le haut comme l'axe Y positif de la scène.
  - **Object Up** - Considère le haut comme le vecteur pointant du transform cible de la contrainte vers le transform spécifié dans la propriété `World Up Object`.
  - **Object Up Rotation** - Considère le haut comme l'axe `World Up Vector` dans l'espace local du transform spécifié dans la propriété `World Up Object`.
  - **Vector** - Considère le haut comme le `World Up Vector` en espace monde.
  - **None** - Ne définit pas de direction vers le haut.
