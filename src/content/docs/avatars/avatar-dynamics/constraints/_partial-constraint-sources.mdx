La liste "Sources" détermine quels transforms affectent cette contrainte. Cliquez sur le bouton `+` en bas à droite pour ajouter une nouvelle source. Cliquez sur le bouton `-` pour supprimer la source actuellement sélectionnée. Chaque source a les options suivantes :
- **Source Transform** - Le transform à utiliser comme source.
- **Weight** - Contrôle l'influence de cette source sur la contrainte.
  - Cela devrait normalement être dans la plage de 0 à 1, mais vous pouvez cocher la case `Free edit` si vous souhaitez définir des valeurs en dehors de cette plage.
