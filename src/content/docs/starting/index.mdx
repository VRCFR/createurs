## Comprendre le Creator Companion

Ce logiciel à était conçu par VRChat pour simplifier l'installation de Unity et du SDK, voici les différents éléments qu'il Comprendr:
- preset de création de projet (World, Avatar, World avec Udon)
- Gestionnaire de paquet Unity pour les mises à jours
- Mise à jour simplifié du SDK de VRChat
- Interface facile d'utilisation
- Possibilité d'ajouter ces propres paquet 

![](/starting/CC.png)

### Les onglets 

Vous avez 3 différents types d'onglets: 
- Project
- Learn
- Tools

#### Project

Sur cette onglet, se trouvera la liste de vos projet Unity créer et importer sur le Creator Companion.

cela vous permettra de gérer vos projets et vos packages.

![](/starting/cc1.png)

Les informations présente sont : **Project Name** *(le nom de votre projet)*, **Project Type** *(Si c'est un monde ou un avatar)* et deux boutons

- **Open Project** : Ouvre le projet dans Unity
- **Manage Project** : Permet de gérer les paquets présent dans le projet
![](/starting/cc3.png)

#### Learn

Sur cette onglet, vous retrouverez des liens vers des tutoriels et des guides pour vous aider à créer votre contenue.
![](/starting/cc4.png)

#### Tools

Actuellement cette section contient que un élément: **VRC Quick Launcher**

![](/starting/cc6.png)

##### VRC Quick Launcher

Ce programme permet de lancer des instances locales de VRChat 

![](/starting/cc5.png)
